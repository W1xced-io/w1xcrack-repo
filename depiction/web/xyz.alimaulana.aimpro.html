AIM Pro

iOS 16 Style Lockscreen

AIM Pro is the perfect version of AIM, now it can be customized on Lockscreen or Notifications Center.

Important: AIM Pro only adds customization on the lockscreen. So I only sell the lockscreen customization feature, I don't sell AIM and as you can see it's free as of now.

Features

Time

  • Custom font
  • Custom font size
  • Custom color
  • Position offset
  • Alignment

Date

  • Custom font
  • Custom font size
  • Custom color
  • Position offset
  • Alignment
  • Date option (Style: uppercase, lowercase, capitalized, none)

Widget

  • Widget option
  • Custom color
  • Position offset

Note: Custom font put in /Library/Application Support/AIMFont (format .ttf/.otf/.woff)

Tip: To change a custom rainbow color on the lockscreen with the color picker, hold on the rainbow color section to display a list of colors.
