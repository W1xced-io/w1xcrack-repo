Pale

A beautiful and minimalistic UI tweak for iOS.

Features

  • Pale status bar loading support

Requirements

  • Snowboard Status Bar Extension + Juice Status Bar support is required for the LTE icon to work
    • Get Snowboard from https://miro92.com/repo/
    • ⚠️ Does not currently support iOS 15/16
  • Dual SIM users: Need to disable secondary cards ⬆ (Please review and confirm before purchasing)

CallAssist

  • 100px perfect support (requires fine-tuning of icon position and size)

Contains:

  • Signal
  • Wi-Fi
  • Network Type (3G, 4G, 5G, LTEx2)
  • Battery (Juice)

Compatibility

Supports iPhone, iPad, and iPod touch running iOS 12.0 to 16.5. Jailbreak required.
